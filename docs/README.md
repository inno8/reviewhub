# ReviewHub

Code review teaching dashboard — transforms daily @code-review findings into structured learning for interns with performance tracking.

## Status

🟡 **Planning** — Design phase

## Team

- **Lead:** Yanick
- **Agents:** @code, @designer, @devops

## Tech Stack

- **Frontend:** Vue.js 3, Tailwind CSS, Vite
- **Backend:** Node.js, Express, TypeScript
- **Database:** PostgreSQL, Prisma
- **Integrations:** GitHub API, Telegram Bot

## Features

- 📅 Calendar-based review navigation
- 🔍 Side-by-side code comparison
- 📚 Teaching explanations (WHY it's better)
- ✅ Mark as understood
- 📞 Request explanation call (Telegram notification)
- 🚀 One-click fix application & PR creation
- 📊 Performance insights per developer
- 📈 Strengths, growth areas, learning recommendations
